# About Game
Just some random game without a name I feel like making. That's right, in one class ;)

# Installing Game
```
git clone https://github.com/Arinerron/Game.git
```

# Starting Game
```
sh run.sh
```

# Controls
- `W` - Move up
- `A` - Move left
- `S` - Move down
- `D` - Move right
- `<SPACE>` - Jump
- `<RIGHT_CLICK>` - Pan and speed walk
- `<SHIFT>` - Slow walk
